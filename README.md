# secred.app

> Secure any data: password, message or link

## Project setup

### Prepare config file

```bash
cp .env.example .env
```

### Install dependencies

```bash
pnpm install
```

### Run dev server

```bash
pnpm dev
```

### Compiles and minifies for production

```bash
pnpm build
```

### Preview compiled production build

```bash
pnpm serve
```
