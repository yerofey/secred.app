# secred.app

> Secure any data: password, message or link

## Project setup

### Prepare config file

```bash
cp .env.example .env
```

### Install dependencies

```bash
# using NPM
npm i

# using PNPM
pnpm i
```

### Run dev server

```bash
# using NPM
npm dev

# using PNPM
pnpm dev
```

### Compiles and minifies for production

```bash
# using NPM
npm build

# using PNPM
pnpm build
```

### Preview compiled production build

```bash
# using NPM
npm serve

# using PNPM
pnpm serve
```
